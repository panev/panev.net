I'm a web designer & front-end developer based in Sofia, Bulgaria. I currently work for Infragistics, a software components vendor, where I create visuals and write code for our web dev tools product – IgniteUI.

Some of the things I do:

Web UI & UX

I specialize in creating user experiences and engineering user interfaces for the web and various types of apps.

Front-end development

I also do Front-end development, I find great joy in giving life to the concepts and various UI/UX solutions I create, tweaking the latest & greatest web technology has to offer.

Branding

In addition to that I build brands and work with people to develop their individual or corporate identities, not only creating a logo symbol, but establishing consistent visual communication across every bit of the user experience.

Helping out:

Occasionally, I do pro–bono consulting & volunteer work for non–profit organizations and causes I care about. If you have a project that aims to better things for humanity, save the environment, explore space or fight fires—and you're in need of advice or assistance—don't hesitate to contact me.

For larger, more commercially orientated entities, I offer end–to–end consulting services on an hourly basis. ;-]
